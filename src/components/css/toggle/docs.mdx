# Toggle
A toggle allows the user to switch an option on or off.

<Canvas mdxSource='<input type="checkbox" class="toggle">'>
  <Story id='css-only-toggle--default'></Story>
</Canvas>

## Examples

### Checked
Use the <code>checked</code> attribute to activate the toggle.
<Canvas mdxSource='<input type="checkbox" class="toggle" checked>'>
  <Story id='css-only-toggle--checked'></Story>
</Canvas>

### Disabled
Use the <code>disabled</code> attribute to disable the toggle.
<Canvas mdxSource='<input type="checkbox" class="toggle" disabled>'>
  <Story id='css-only-toggle--disabled'></Story>
</Canvas>

### Custom Size
Use the available custom properties to make the switch a different size.
<Canvas mdxSource='<input type="checkbox" class="toggle" style="--chkbs:2rem;">'>
  <Story id='css-only-toggle--custom-size'></Story>
</Canvas>
