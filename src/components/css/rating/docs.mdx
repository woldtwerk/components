# Rating
Ratings give users a way to quickly view and provide feedback.

## Examples

### Maximum Value
To change the maximum possible value, use the <code>max</code> attribute.

### Steps
Use the <code>steps</code> attribute to let users select fractional ratings. Not recommended for input.

### Disabled
Use the <code>disabled</code> attribute to disable the toggle.

### Custom Size
Use the available custom properties to make the switch a different size.


### Importing
